SGA Dashboard Final

This package is wired to the 7 published CSV feeds from the Google Sheet.

Included:
- Projects tab
- Workload tab
- Financial tab
- Archive toggle
- Sort by job number / job name
- Billing health badge next to status
- Projects / financials / consultants / AR / AP / revenue history / tax summary wired live

Deploy:
1. Replace your repo root files with this package contents.
2. Keep package.json, vite.config.js, index.html, and src/ at repo root.
3. Delete old tailwind.config.js and postcss.config.js if they still exist.
4. Commit to main.
5. Vercel auto-deploys.
